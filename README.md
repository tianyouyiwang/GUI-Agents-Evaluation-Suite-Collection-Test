# GUI Agents Evaluation Suite Collection

This repository collects information about the HuggingFace evaluation suite for GUI Agents, focusing on desktop environment-related Grounding benchmarks.

## Desktop Environment Grounding Benchmarks

- ScreenSpot: https://huggingface.co/datasets/osworld/ScreenSpot
- ScreenPoint: https://huggingface.co/datasets/osworld/ScreenPoint
- ScreenPrism: https://huggingface.co/datasets/osworld/ScreenPrism
- SeeClick: https://huggingface.co/datasets/osworld/SeeClick

## Overview

The HuggingFace OSWorld evaluation suite provides standardized benchmarks for evaluating GUI agents in desktop environments. The grounding benchmarks above focus on evaluating how well agents can locate and interact with specific UI elements on screen.
